# 数据来源
* 2015-12-10到2018-11-16，42万条911紧急电话的数据
* 数据来自[Kaggle](https://www.kaggle.com/mchirico/montcoalert/data)

# 分析目的
* 统计紧急电话的不同事件类型的次数
* 统计不同月份紧急电话的事件类型的变化

# 字段含义
* title：报案事件分类
    * EMS：紧急突发情况
    * Fire：火灾
    * Traffic：交通事故

* lat：维度
* lng：经度
* desc：报案内容描述
* zip：
* timeStamp：时间戳
* twp：报案地点-小镇、县城名称
* addr
* e
